Introduction

The key differences between Postmark and MailPace

This guide is an easy to use reference to help you migrate from Postmark to MailPace. It covers the differences between the MailPace and Postmark APIs, how to send outbound emails and process inbound emails, webhooks and more.

There are a number of important differences to be aware of when migrating from Postmark to MailPace, namely:

  • Postmark is owned by an email marketing company, ActiveCampaign which also provides Marketing email. MailPace only supports Transactional email, and is an independent company
  • Postmark supports marketing emails, MailPace only allows transactional email, and has no support for marketing email
